## SOMA Crédito - Landing de Vendas

Landing page em React + Vite + TypeScript + Tailwind, focada na venda de produto de crédito com CTA para WhatsApp.

- Stack: React, Vite, TypeScript, Tailwind CSS
- Seções: Navbar, Hero, Benefícios, Como funciona, Oferta, FAQ, CTA final, Footer
- CTA principal: botão de WhatsApp usando `WHATSAPP_URL` definido em `src/config/whatsapp.ts`

### Scripts básicos

- `npm install` — instala dependências
- `npm run dev` — sobe o ambiente de desenvolvimento
- `npm run build` — build de produção
